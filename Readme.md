節能習慣追蹤器 Energy Saver Tracker

一款以 SDG 7 / 11 / 13 為理念設計的網頁工具，
讓使用者透過每日節能行為累積點數、查看歷史紀錄、並在每週達標後獲得獎牌獎勵。

📌 功能特色
🌱 1. 每日節能行為紀錄

15 項可勾選的節能行為（如隨手關燈、減少冷氣使用等）

點擊可累積相對應點數

同一項目每日僅能紀錄一次

附加「每日備註」欄位可手寫心得

🎯 2. 每週進度計算

每週目標為 100 點

週一 00:00 自動重置

走路小人動畫根據本週進度自動移動

📝 3. 歷史紀錄（可分頁瀏覽）

自動儲存每日：日期、行為列表、獲得點數、備註

以 LocalStorage 儲存，重整不會消失

🏅 4. 獎牌系統

本週累積分數達 100 點自動獲得獎牌

附帶隨機一則鼓勵語

所有獎牌可於「獎牌紀錄」頁查看

🔊 5. 儲存音效

紀錄時播放清脆提示音
